Task List Manager

Welcome

This is a task list application. Neat, right? You can write down tasks, set a deadline and more. Then click the checkbox when you're done with a task. It should contain the title of your task, deadline (date and time), the priority of the task (High, medium, low) and a description explaining that the task is about.

Each task get a colored border based on how much time is left. Green ones are completed, red ones are expired.
